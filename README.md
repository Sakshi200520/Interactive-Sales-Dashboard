# Interactive-Sales-Dashboard
Developed a dynamic dashboard using Streamlit and Plotly to analyze sales data with filters, KPIs, and interactive visualizations for better business insights.
##  Features-
 -Interactive filters (Product & Region)- 
 -KPI metrics:  - Total Revenue  - Total Orders  - Average Order Value- 
 -Visualizations:  - Line Chart (Sales Trend)  - Bar Chart (Product-wise Revenue)  - Pie Chart (Region-wise Sales)  - Scatter Plot (Quantity vs Price)- 
 -Statistical Analysis:  - Histogram (Sales Distribution)  - Box Plot  - Violin Plot-
 -Correlation Heatmap- 
 -Download filtered dataset- 
 -Clean and responsive dashboard layout---## 
 -Tech Stack- Python  - Streamlit  - Plotly  - Seaborn  - Matplotlib  - Pandas  - NumPy  ---##
